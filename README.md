# Simple snake rainbow loader

## Install

```js

  npm install snake-cli-loader -g

```

## Usage

```js
        
      var snakeloader = require('snake-cli-loader');
      
      snakeloader.start(); 
      

      // Sometime later;
      snakeloader.stop();


```
